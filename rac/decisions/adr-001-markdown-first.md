# ADR-001 Markdown First

## Status

Accepted

## Context

RAC needs a single canonical source format for requirements and other artifacts.

## Decision

Markdown is the canonical source format.

## Consequences

### Positive

- Human-readable
- Git-friendly
- Tool-independent
